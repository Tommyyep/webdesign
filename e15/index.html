1. Open XD
  2. Create a new artboard that is 600px by 600px
  3. Select the rectangle tool
  4. Create a 42px(width) by 316px(height) rectangle at x300 y101
  5. The shapes can be created anywhere first and later placed at the directed location using the cursor tool if needed
  6. Create a 138px by 34px rectangle at x252 y417
  7. Create a 42px by 99px rectangle at x300 y451
  8. Select the triangle tool
  9. Create a 42px by 57px triangle at x300 y44
  10. Select the circle tool
  11. Create a 30px by 30px circle at x306 y421
  12. Select the cursor tool
  13. Selecting the triangle and top rectangle using the cursor tool, uncheck the Border box and set the fill color to #EDEDED
  14. Selectiing the bottom 2 rectangles using the cursor tool, uncheck the Boder box and set the fill color to #FFEE7C
  15. Selecting the circle using the cursor tool, uncheck the Border box and set the fill to #FF0000
  16. export as a png
